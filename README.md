# Proyecto-Manipuladores-Roboticos
Repositorio documentos proyecto manipuladores
#Descripción
- Este proyecto consiste en desarrollar un brazo tipo SCARA para detección y clasificación de pernos y tornillos, haciendo uso de OpenCV con python donde se realizara todo el procesamiento pesado y posteriormente se controlara el robot con arduino
#Componentes
- 02 servomotores MG996R
- 01 servmotor SG90
- Laptop o pc con VS code
- Arduino
- 01 iman
- 01 resistencia de 330
- 01 resistencia 5k
- 1 Mosfet Tipo N IRLZ44
- 01 Diodo
